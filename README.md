# sing-box-hardened

`sing-box-hardened` 是一个一键部署脚本，目标是快速生成基于 **sing-box + Argo 隧道 + 多协议** 的可用节点配置，并提供订阅/模板导出能力。

## 项目说明

- 本项目基于并参考上游项目：**[fscarmen/sing-box](https://github.com/fscarmen/sing-box)**
- 在上游优秀工作的基础上，结合本仓库的需求做了增强与定制化调整（例如菜单项、配置生成策略与交互流程等）

如果你想了解原始设计思路、更多发行兼容细节、社区讨论与更新节奏，建议优先阅读上游仓库文档。

## 主要功能

- 一键安装与管理 sing-box
- 集成 Argo 隧道（含临时隧道、Token/Json/API 方式）
- 多协议快速生成（按脚本菜单可选）
- 支持客户端配置导出与订阅模板
- 支持常见运维动作：查看链接、换端口、增删协议、切换 Argo、更新内核等

## 快速开始

### 1) 下载并执行

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/PPX-LuBing/sing-box-hardened/main/sing-box-hardened.sh)
```

### 2) 运行后按菜单操作

安装完成后，可通过脚本内创建的快捷命令进行管理（通常为 `sb`）。

常见操作包括：

- 查看节点信息
- 更换监听端口
- 同步 sing-box 版本
- 增加/删除协议
- 更换 Argo 隧道
- 更换 CDN / 优选域名

## 使用建议

- 建议使用全新系统或最小化环境部署，避免与已有代理服务冲突
- 修改端口前先确认端口未被占用
- 生产环境建议保留配置备份，并分批次调整参数
- 若启用了云厂商防火墙/安全组，请同步放通对应端口
